# Changelog

## 0.1.0

- Initial release.
- `URL to Markdown` node: fetch a URL and convert the page to clean Markdown.
- Readability extraction (strip nav/ads/footers), GitHub-flavored Markdown via Turndown, markdownlint auto-fix.
- Options: destination field, Readability toggle, lint toggle, heading style, bullet marker, code block style, request timeout, custom User-Agent.
- Exposed as an AI Agent tool (`usableAsTool`).
